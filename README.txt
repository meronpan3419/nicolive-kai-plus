■概要
豆ライブ改プラスは、豆ライブの派生の豆ライブ改の派生です。
詳細は上記の豆ライブと豆ライブ改の配布サイトから御覧ください。
非力なパソコンでも配信できるツールなのですが、追加機能があるのでどうなのでしょう。


■免責
本ソフトウェアを使用した事により発生した損害について、一切の責任を負いかねますので。


■動作環境
-.net framework3.5以降でいいはず
-Flah Player ブラウザで通常のニコ生配信が出来るバージョンならたぶんいけるはず
-InternetExplorer7以降(保護モードなんたらでieframe.dllが必要なのでIE6以前は利用不可)
-OS Windows 7以降（windows XPのサポート終了のため7以降を推奨。Vistaとは？）
-CPU Intel Atom N270相当以上
-メモリ 1GB以上

ネットブックでも配信できないことは無いですが、外部配信ツールを使用する場合、それに合わせた環境が必要かもしれません。


■インストール
nicolive-kai-p*.rarを解凍しするだけ。NicoLive.exeが本体の実行ファイルです。


■アンインストール
レジストリはいじってません。
解凍先のフォルダと下記の設定ファイルを削除すればいいはずです。
WindowsXP だと以下のフォルダに設定ファイルが保存されています。
%UserProfile%\Local Settings\NicoLive\
Vista/7もそれ相当のファルダに保存されてます。


■追加機能
○全般
- ステータスバーに「枠数/枠待」項目
- 配信コンソール部にコメント欄、通常/外部配信・FMEプロファイル切り替え欄
- F2を何回か押しておけば前枠使いまわして枠取れる簡単枠取り機能
- IE7以降で保護モードでもクッキーをIEのものに上書き
- コミュ限の時はタイトルバーの表示
- 豆ライブの起動オプションにfastliveを追加しておくと起動直後に自動的に枠とって配信する

○コメビュ
- コメントサーバーとのコネクションチェック強化＋再接続
- Ctrl+F12で接続キャンセル
- 再接続時のコメント読み込み・コメビュ表示高速化
- コメビュの「NGコメントです」表示と「>> NGです」コメント
- コメビュ表示項目「時間」「情報（運営コメ 公式コメ BPS 携帯 プレミアム）」欄
- コメビュ色付け
- フォント選べる
- NGユーザー登録
- コメントを選んでrキーを押すとアンカー！
- コメントを選んでctrl+wでコメントパクる！
- コメントを選んでctrl+eでアンカーつけてパクる！
-　ctrl+qで棒読みちゃんの読み上げキューをクリア（嵐対策に）。
- vimの移動コマンドみたいにコメビュ移動(ctrl+jでコメビュ下に、ctrl+kでコメビュ上に、ctrl+nで一番新しいコメントに移動)
- 自動コテハン取得用の正規表現をいじれるように（突然の運営の仕様変更でもたぶん平気）
- コメントくれた人に「いらっしゃい」言う機能
- 「おみくじ」に反応して運勢占う機能（配信たん２からインスパイアされて）
- 「相性」に反応して相性占う機能（配信たん２からインスパイアされて）
- 184の人に勝手にコテハンを付ける機能
- フラッシュコメントジェネレーターに対応。

○今ココなう！
- 現在地定期コメント
- ジオイド補正されてないGPS用の高度補正（@ALTITUDE＿EGM96 http://earth-info.nga.mil/GandG/wgs84/gravitymod/egm96/binary/WW15MGH.DAC をnicolive.exeと同じフォルダにポイッ）

○ツイッター
- 配信開始ツイートで@TITLE出ないように。
- ハッシュダグを指定出来るように。

○外部配信
- FMEプロファイル選択
- NLE対応
- XSplitショートカットキー(とりあえずctrl+alt+A)で配信開始・停止

○読み上げ
- コテハン、主コメ、/コメ、#コメ ON/OFF

○簡易ビューアー
- 小さく出来るように。
- 最前面表示できるように。
- 画面位置を覚えておくように。
- 放送接続時に自動起動

○ランチャー
- 放送接続時に外部ツール起動
- 引数に＠LVで放送IDに置換
