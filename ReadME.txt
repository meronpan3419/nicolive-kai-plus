NicoLive-1.0.46-no-iTunes-改27使い方


(1). Flash Player の最新版をインストールして下さい。
(2). NicoLive改をインストールしたいディレクトリの下に、このアーカイブ内の
　　 *.exe および *.dll ファイルをすべてコピーします。既存のNicoLiveに上書き
　　してもかまいません。
(3). 初期設定の内容を設定しなおしてください
　　（多分クリアされているはずなので）。
(4). prop_history.xml はオリジナルの豆ライブとフォーマットが変わって
　　いるため、消しておいたほうが良いでしょう。

・FMEステータスでの、「再起動」の確実性の向上
・FME配信中に、定期的(11分ごと)に席替えを行う機能
・ログアウトした際に、自動再接続する機能を追加(kai2)
・184を付けていない人のコテハンを自動取得する機能復活(kai3)
・開演待ちのスキップを飛ばす機能をつけました(kai5)
・来場者数の更新、コテハン取得のバグ修正(kai6)
・今ココで現在地を行進している場合、「現在地は？」で現在地を
　答えます。(kai8)
・「前枠の続き」機能で、タグを引き継ぐようにした(kai9)
・FMEをGUIモードで動かす機能を付与した(kai10)
・「速度は？」と聞くと、速度を答えるようにした(kai10)
・「前枠の続き」を2.0.2からバックポート(kai12)
・ラグ対策の席替え機能を、FMEだけでなく通常配信でも行うようにした(kai14)
・別ウインドウのコメントのコテハンの不具合を修正(kai15)
・ステータスバーの表示項目を設定する(kai16)
・FMEの起動に失敗する問題を修正(kai16b)
・FMEの設定の表示をon/offできるようにした(kai17)
・FME GUIの起動がうまくいかない問題と、簡易ビュアの問題を解決(kai18)
・ラグ対策を席替えからリロードに変更(kai19)
・簡易ビューアのSWFのプレイヤーをもとに戻した(kai20)
・再接続時のコメントを重複して音声出力するのを直した(kai20)
・ブラウザのキャッシュを参照して、セッション情報を用いて接続可能にした(kai21)
・前のタイトルが
　　その１、その(1)、 ほにゃらら(1)、＃1、などで終了している場合、
　前枠の続きや、連続枠取りで、番号部分がインクリメントされます。(kai22)
・F4(ファンクションキー４）を押すと現れるコメントウインドウのステータス
　バーの一番左側に、放送の経過時間を表示するようにした(kai23)
・連続枠取り等がonとなっている場合の表示を見やすくした(kai24)
・簡易ビューアのサイズを若干変更(kai25)
・FME配信のfmesessons.datを消す仕様を付加した。FMEタブのfmesessions.dat(kai25)
　の設定をWindows7以外の人は変更すること。(kai25)
・「現在地は？」「速度は？」の読み上げのロジックを修正(kai25)
・自動無料延長機能のfix(kai26)
・前枠の続きで、クルーズ設定を追加(kai26)
・前枠の続きで、クルーズ設定がうまく読めない問題をfix(kai27)
